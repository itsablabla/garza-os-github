# Beeper Fast MCP

Lightweight Python MCP server for Beeper messaging - optimized for speed.

## Features
- Search chats and messages
- Send messages to conversations
- List recent chats
- Get chat details

## Deployment
Fly.io (`beeper-fast-mcp`)

## Environment Variables
- `BEEPER_TOKEN` - Beeper API token
- `API_KEY` - Server authentication key

## Files
- `main.py` - Server implementation
- `fly.toml` - Fly.io config
- `Dockerfile` - Container build
