# Computer Commander MCP

Local Docker container control for Computer Use instances.

## Features
- Start/stop Computer Use containers
- Get container status
- Execute commands in containers
- View container logs
- Get access URLs

## Deployment
Local (Mac Mini)

## Environment Variables
- `CC_API_KEY` - Server authentication key
- `PORT` - Server port (default: 3100)

## Container Management
Controls `computer-commander` and `computer-commander-2` Docker containers.

## Tech Stack
- Express.js
- Docker CLI integration
