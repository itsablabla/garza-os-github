# Abode MCP

MCP server for Abode home security system control.

## Features
- Get/set alarm mode (standby, home, away)
- List devices and their states
- Lock/unlock smart locks
- Switch devices on/off
- Trigger automations

## Deployment
Local Python server

## Environment Variables
- `ABODE_USERNAME` - Abode account email
- `ABODE_PASSWORD` - Abode account password

## Usage
```python
python server.py
```
